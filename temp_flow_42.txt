1. client joins
2. client uses local initialization
3. Local Initialization calls Server to create new DataController
4. DC gets instanced in server.
	4.1 If no previous moddata has been found, create new one for player
	4.2 Otherwise, get old moddata and applies to instance of DC
5. Sync with Client, sends DC data through GlobalModData transmit to specific Client
6. Client requests Cache, force recalc from server
7. Server recalc cache
8. Server sends cache to client
9. Client receives cache
